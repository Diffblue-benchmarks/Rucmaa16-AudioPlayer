Informationen zu Lautstärkeregler

Damit der Lautstärkeregler funktioniert, bitte die nircmd.exe Datei ausführen.

1)	nircmd.exe als Administrator ausführen
2)      Im Dialog auf "Copy to Windows Directory" klicken
3) 	Im Dialog auf "OK" klicken

Der Lautstärkeregler kann nun verwendet werden
